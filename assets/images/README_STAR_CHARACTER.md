# Image de l'Étoile pour l'Écran de Félicitation

## Emplacement
Placez l'image de l'étoile (deuxième image en pièce jointe) dans :
```
assets/images/star-character.png
```

## Description
Cette image représente une étoile dorée animée et joyeuse, utilisée dans l'écran de félicitation lors du passage au chapitre suivant.

## Format
- Format recommandé : PNG avec transparence
- Taille recommandée : 200x200px (sera redimensionnée automatiquement)
- Fond : Transparent

## Utilisation
L'image est utilisée dans `src/screens/ChapterCompletion/index.js` pour afficher l'étoile animée lors de la complétion d'un chapitre.
