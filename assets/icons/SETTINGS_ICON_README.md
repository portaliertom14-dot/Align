# Logo des Paramètres

## Instructions

Pour intégrer le nouveau logo des paramètres :

1. Placez votre fichier PNG dans ce dossier (`assets/icons/`)
2. Nommez-le exactement : `settings.png`
3. Dimensions recommandées : 24x24px ou 48x48px (pour la haute résolution)
4. Format : PNG avec transparence (fond transparent)

Le logo sera automatiquement utilisé dans :
- `src/components/Header.js` (bouton paramètres dans le header)
- `src/screens/Feed/index.js` (bouton paramètres sur l'écran d'accueil)

## Note technique

Le code utilise `tintColor: '#FFFFFF'` pour appliquer une teinte blanche à l'image. Si votre logo est déjà blanc ou si vous préférez garder les couleurs originales, vous pouvez retirer la propriété `tintColor` dans les styles.
